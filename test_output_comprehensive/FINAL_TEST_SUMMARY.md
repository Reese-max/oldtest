### 🟡 中等優先級: 格式自動檢測

#### 實施方案

```python
class FormatDetector:
    """自動檢測題目格式類型"""

    def detect(self, text):
        # 檢測選項標籤
        has_labels = bool(re.search(r'\([A-D]\)', text))

        # 檢測題號格式
        has_dot = bool(re.search(r'^\d+\.', text, re.MULTILINE))

        return {
            'format': 'labeled' if has_labels else 'unlabeled',
            'number_style': 'dot' if has_dot else 'space'
        }
```
